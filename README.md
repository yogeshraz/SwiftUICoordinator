SwiftUICoordinator is a native, declarative framework designed to facilitate seamless programmatic navigation within SwiftUI applications.
## Requirements

| Platform | Minimum Version |
|----------|-----------------|
| iOS      | 17.0            |
| macOS    | 13.0            |
| tvOS     | 16.0            |
| watchOS  | 9.0             |

<br>


## Installation

You can install `Routing` using the Swift Package Manager.

1. In Xcode, select `File` > `Add Package Dependencies`.

  <img src = "https://github.com/JamesSedlacek/Routing/blob/main/Assets/Installation1.png" width="350">
</p>
<br>

2. Copy & paste the following into the `Search or Enter Package URL` search bar.
```
https://github.com/yogeshraz/SwiftUICoordinator.git
```
